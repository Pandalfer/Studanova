# Studanova

Studanova is a productivity tool for students built with [Next.js](https://nextjs.org).

## Technical Features

=> Page skeletons: Prevents layout shift and improves perceived load times during data fetching.
=> UUIDs: Ensures unique identification across the database and prevents ID enumeration attacks.
=> dnd-kit functionality: Enables intuitive drag-and-drop organization for notes and folders.
=> Database error handling: Manages server failures gracefully to ensure UI stability.
=> Recursive logic: Powers the folder tree to support infinite nesting and deep data traversal.
=> DOMPurify: Sanitizes HTML input to protect the application from Cross-Site Scripting (XSS).
=> Toasts: Delivers real-time, non-intrusive feedback for background actions and status updates.
=> Argon2 hashing: Implements high-grade password encryption to ensure user data security.
=> Custom hooks: Abstracts complex state and logic to maintain a clean, modular codebase.
=> Custom toolbar: Provides a tailored rich-text editing experience with specific formatting tools.
=> Debounced auto-save: Optimizes performance by saving changes only after user input pauses.
=> Levenshtein distance formula: A hand-coded algorithm for intelligent, typo-tolerant search results.
